# Support

This repository is for the standalone CairnID public site.

## Site Issues

Use this repository for site-only issues such as:

- Broken site pages, routes, links, or navigation.
- Site build failures or dependency-lock problems.
- Accessibility, layout, or content regressions in the public documentation surface.
- Documentation corrections that apply only to the site.

Do not include secrets, tokens, keys, private deployment data, or sensitive logs in public issues.

## Product And Security Reports

Product runtime questions, deployment behavior, authentication behavior, API behavior, and operator reports are outside this site repository's support scope. Use the product's published docs or process for those questions and reports.

Do not open public issues for vulnerabilities. For site-repository vulnerabilities, use GitHub private vulnerability reporting as described in [SECURITY.md](SECURITY.md).

Do not use this site repository to report product/runtime vulnerabilities.

## Support Channels

Use only support channels listed in public repository or organization settings/docs. Do not send private data through public issues.
